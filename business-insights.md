# Business Insights

## Summary

This analysis reviews clean sales performance and separates rows that need data quality review before they are used in business decisions.

## Key Findings

1. **Revenue Focus**

   Electronics generated the highest clean revenue. This category should be reviewed further to understand which products, channels, or regions are driving performance.

2. **Profitability**

   Stationery had the strongest profit margin. This suggests the business should compare revenue size with margin before deciding which category deserves more focus.

3. **Regional Performance**

   Makassar was the strongest region by clean revenue. This region can be used as a benchmark when reviewing lower-performing regions.

4. **Data Quality**

   114 rows needed review before dashboarding. This matters because missing values, duplicate orders, and non-completed statuses can distort the final metrics.

## Recommendation

Use clean rows for the main dashboard, but keep the issue log visible. The business should review non-clean rows before using them for final reporting, especially if they affect revenue, profit, or order completion.
